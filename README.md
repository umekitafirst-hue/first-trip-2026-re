# 社員旅行2026 〜佐賀県編〜 🗾

株式会社FIRST 社員旅行用モバイルアプリ

## 機能
- 📅 スケジュール（しおりデザイン・メモ編集）
- 🗺️ 旅程・連絡先
- 🏨 ホテル情報・館内施設
- 🌊 観光スポット
- ✅ 事前準備チェックリスト
- 🛏️ 部屋割り（8色カラー分け）

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## デプロイ

`main` ブランチへのプッシュで GitHub Pages に自動デプロイされます。

### GitHub Pages の有効化手順
1. リポジトリの **Settings** → **Pages**
2. Source を **GitHub Actions** に設定
3. `main` ブランチにプッシュするだけで自動デプロイ ✅
